# CRUD en Laravel


<p>CRUD de usuarios realizado en Laravel v5.7 con autenticación de Laravel. </p>
<p>Datos que se obtiene de los usuarios:</p>
<ul>
  <li>Nombre</li>
  <li>Apellido</li>
  <li>Correo</li>
  <li>Teléfono</li>
  <li>Móvil</li>
  <li>Dirección</li>
  <li>Ciudad o Municipio</li>
</ul>
